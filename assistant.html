Assistant Linux

Outil CLI pour guider les débutants dans l’univers Linux

Objectif

L’assistant Linux est un outil CLI permettant d’apprendre les bases de Linux via des commandes interactives. Il installe automatiquement les outils essentiels et guide l’utilisateur pas à pas dans l’apprentissage.

  • Installation automatisée d’outils (git, python, docker, etc.)
  • Scripts explicatifs et démonstrations en direct
  • Approche pédagogique pour étudiants et autodidactes
