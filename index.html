Antares — Logiciel de chiffrement macOS

Description : application desktop (Go + Fyne) pour chiffrer/déchiffrer des dossiers. Fonctionnalités principales :

  • Argon2id pour dérivation de clé, AES-128/256 selectable
  • Header de fichier contenant niveau de chiffrement, IV, tag d’intégrité
  • Intégration Touch ID / Secure Enclave pour déverrouillage
  • Mode debug sécurisé (logs chiffrés)

Code : privé — contact par mail si accès demandé.

C-grid — PFE

Résumé : Conception d’un algorithme symétrique basé sur l’analyse fréquentielle des bigrammes disposés dans des grilles multidimensionnelles. Objectifs : résistance à l'analyse statistique, vectorisation et optimisation performance.

Livrables : rapport PFE (PDF), slides, code prototype (privé/public selon version), benchmarks comparatifs AES vs C-grid (petits textes).

Ressources : screenshot, diagramme d’architecture et lien vers le PDF (si tu veux l’héberger dans /assets/docs/).

detect-malicious-url

Outil web (frontend + backend) exécutant plusieurs analyses sur une URL soumise :

  • Vérification heuristique (domain age, hosting)
  • Analyse statique du contenu (patterns stéganographiques connus)
  • Sandboxing / preview (capture d’écran) — si disponible

Démo vidéo / screenshots à ajouter ici. Le service public n’expose pas d’API de suppression/scan automatique (sécurité et éthique).
