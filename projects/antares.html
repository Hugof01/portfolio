Antares

Logiciel de chiffrement macOS sécurisé (Touch ID / Secure Enclave)

Présentation

Antares est un logiciel macOS développé en Go avec le framework Fyne. Il permet de chiffrer et déchiffrer des dossiers via un système modulaire et sécurisé, intégrant Touch ID et la Secure Enclave d’Apple.

Fonctionnalités principales

  • Dérivation de clé via Argon2id (remplace SHA-256 classique)
  • Chiffrement AES-128 ou AES-256 selon le niveau choisi
  • Header sécurisé : IV, tag d’intégrité, version du format
  • Intégration Touch ID / Secure Enclave
  • Mode Debug sécurisé (logs chiffrés et signés)
